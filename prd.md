# Interactive Resume Website MVP – PRD

## Project Overview
This project is a single-page web application that serves as an interactive resume. It presents my skills, experience, certifications, and projects in a clean, modern format that is more engaging than a traditional resume.

## Target User
- Recruiters and hiring managers
- Employers in IT and cybersecurity

## Problem
Traditional resumes are static and do not effectively showcase technical skills, projects, or real-world experience.

## Core Features

### 1. Resume Section Display
Displays structured sections including summary, skills, certifications, and experience  
Success: Users can quickly understand qualifications

### 2. Security Case Study Section
Highlights a real-world vulnerability disclosure project  
Success: Demonstrates practical cybersecurity experience

### 3. Print-to-PDF Feature
Allows users to print or save the resume as a PDF  
Success: Resume can be easily shared offline

## User Experience Flow
1. User opens the website  
2. Views header with name and contact info  
3. Scrolls through sections (skills, experience, case study)  
4. Uses print button if needed  

## Technical Constraints
- Single HTML file  
- HTML, CSS, JavaScript only  
- No backend  
- Runs in browser  

## Out of Scope
- User accounts  
- Backend database  
- Job application system
