# Interactive Resume Website MVP

## What It Does
This is an interactive resume website that showcases my skills, experience, and cybersecurity work in a modern web format.

## How to Use
1. Open the site
2. Scroll through sections
3. View experience, skills, and projects

## Features
- Structured resume layout
- Security case study section
- CTF competition highlights
- Print-to-PDF functionality

## Live Site
https://resume.taylorjackson.io

## What I Learned
I learned how to use AI tools to rapidly prototype and refine a web application, as well as how to improve layout, usability, and presentation through iteration.
